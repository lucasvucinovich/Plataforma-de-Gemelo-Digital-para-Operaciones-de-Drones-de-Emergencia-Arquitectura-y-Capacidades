# Klout – Digital Twin Platform for Emergency Drone Operations

Klout is a Digital Twin platform designed to support emergency drone operations under pressure.
This project was developed as part of the **HP Semifinal Challenge Competition (HPE CDS Tech Challenge Phase II)**.

The platform provides a unified operational view, real-time telemetry handling, anomaly detection, scenario simulation, and AI-assisted decision support for emergency response coordination.

## Project Overview

Klout demonstrates:

* A unified in-memory Digital Twin model
* Real-time drone telemetry ingestion
* Anomaly detection (battery, communications, coverage)
* Scenario simulation with before/after comparison
* AI-powered decision support (Jarvis)
* Resource coordination and command lifecycle tracking
* Evaluator-ready architecture and demo tooling

The system is built as a full-stack application using **Next.js, React, TypeScript, and Tailwind CSS**, with a modular domain architecture and automated testing across unit, integration, and E2E levels.

## How to Run

Make sure you have **Node.js** installed.

```bash
npm install
npm run dev
```

Then open:

```
http://localhost:3000
```

> To enable the AI assistant (Jarvis), configure the required environment variables as shown in `.env.example`.
